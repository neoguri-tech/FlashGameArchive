# Number Palace Concept Assets

Preview assets for the palace thief art direction. These are not wired into the game yet.

- `palace-track-background.png`: fullscreen palace corridor / centered runner track concept.
- `thief-runner.png`: transparent player character cutout.
- `cipher-gem-tile.png`: transparent blank number-gem tile. Draw digits in canvas/code over this.
- `secret-gate.png`: transparent empty-lane gate for hint/boost choices.
- `crown-shard.png`: transparent score/reward icon.
- `gameplay-style-mockup.png`: overall visual direction mockup, not intended as a direct game asset.

Raw chroma-key sources are kept in `raw/` for regeneration or cleanup.
