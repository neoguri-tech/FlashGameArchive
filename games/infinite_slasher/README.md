# Infinite Slasher / 인피니트 슬래셔

벽이 계속 밀려오고, 벽에 난 틈의 시작점과 끝점을 검선으로 이어 베는 속독 액션 프로토타입.

## 현재 MVP

- 가로, 세로, 대각 변 2연키, 마름모 꺾임, 마름모 한 바퀴, Z자, 4자형 검선 패턴
- 방향키/WASD 입력
- 로비에서 4종 수련장 배경 선택
- 빨간 선의 밝은 시작점부터 `좌 → 상 → 우 → 하`처럼 4방향 점을 연속 입력
- 밝은 타겟 마커는 지금 누를 점, 마젠타 화살선은 다음 진행 방향을 표시
- 모바일/터치용 4방향 패드
- 콤보, 점수, 로컬 최고 기록
- 연속 성공 시 중앙에 뜨는 `N COMBO` 보상 이펙트
- 변을 하나 이을 때마다 화면 끝까지 뻗는 검광 이펙트
- 변 완성 방향에 맞춰 짧게 나오는 8방향 뒷모습 후속 베기 포즈
- 성공 시 벽이 갈라지고 석판 파편이 터진 뒤 다음 벽으로 즉시 전환
- CC0 실제 효과음 샘플 기반의 칼바람, 돌 충격, 석판 파괴 사운드
- PNG 자산 기반의 훈련 석판, 로비용 앞모습 검객, 플레이용 뒷모습 검객 비주얼
- 홈 카드에는 원본 썸네일을 유지한 채 경량화한 `assets/thumbnail-card.jpg` 사용

## 기획 의도

무한의 계단처럼 명시적인 버튼 지시가 아니라 화면 안의 구조를 읽게 하는 것이 목표다. 화살표 시퀀스를 외우는 게임이 아니라, 벽의 빨간 검선을 보고 밝은 시작점부터 다음 점을 빠르게 이어 찍는 게임으로 잡았다.

정밀 리듬 판정은 넣지 않는다. 대신 벽이 가까워지는 압박, 짧은 연속 입력, 연속 격파 사운드와 화면 흔들림으로 리듬감을 만든다.
