# Reversi Chaos Balance Guidelines

이 문서는 리버시 초능력/특수돌 밸런싱을 다시 진행할 때 유지해야 할 기준을 정리한다.
목표는 모든 효과를 얌전하게 만드는 것이 아니라, 파격적인 재미를 유지하면서도 "배정받는 순간 승부가 기우는" 조합을 막는 것이다.

## Core Principles

- 재미가 1순위다. 후보 단계에서는 사기처럼 보이는 파격 효과도 넣고, self-play와 금지 조합으로 거른다.
- 밸런싱에는 시간이 오래 걸려도 된다. 빠른 감으로 확정하지 말고 여러 seed, 랜덤 매칭, 집중 검증을 반복한다.
- 개별 초능력/특수돌만 보지 않는다. 실제 게임은 초능력 1개 + 특수돌 1개 조합이므로 로드아웃 단위로 본다.
- 강한 조합은 바로 효과를 죽이지 않는다. 효과 자체가 재미있다면 해당 초능력+특수돌 조합만 랜덤 배정에서 막는 방식을 우선한다.
- 약한 조합도 문제다. 유저가 받았을 때 "이건 안 쓰는 게 낫다"는 느낌이 들면 좋은 밸런스가 아니다.
- 직관성을 밸런스의 일부로 본다. 텍스트를 곰곰이 읽어야만 이해되는 효과는 실제 플레이에서 약하거나 답답하게 느껴질 수 있다.
- 발동감이 있어야 한다. 효과가 강해도 보드에서 무엇이 바뀌었는지 바로 보이지 않으면 재미가 줄어든다.

## Assignment Rules

- 한 판에는 서로 다른 초능력/특수돌 4개가 등장해야 한다.
- 두 플레이어가 같은 초능력을 갖거나 같은 특수돌을 갖는 배정은 피한다.
- 강력한 후보끼리 붙는 것은 허용할 수 있지만, 조합 승률이 과도하게 튀면 금지 조합으로 막는다.
- 랜덤 배정 풀은 `allowed loadouts`만 사용한다.
- `bannedLoadouts`는 너프보다 먼저 고려하는 안전장치다.

## Primary Metrics

최소한 아래 지표를 함께 본다.

- 로드아웃별 승률
- 로드아웃별 평균 점수차
- 초능력별 집계 승률
- 특수돌별 집계 승률
- 선공/후공 승률 편차
- 초능력별 흑/백 분리 승률과 scoreRate
- 특수돌별 흑/백 분리 승률과 scoreRate
- `sideDelta = blackScoreRate - whiteScoreRate`
- blowout rate: 큰 점수차로 터지는 비율
- 초능력 사용률
- 특수돌 사용률
- 사용했을 때 승률
- 사용하지 않았을 때 승률
- 사용 직후 평균 점수차 변화
- ablation lift: 같은 seed에서 해당 초능력/특수돌만 끈 경우와 비교한 승점/점수차 변화

## Usage-Aware Rule

"배정받은 플레이어의 승률"만 보고 밸런스를 판단하지 않는다.

예를 들어 어떤 초능력을 받은 플레이어의 승률이 50%여도, 실제로 그 초능력을 쓴 판의 승률이 낮고 안 쓴 판의 승률이 높다면 그 초능력은 좋은 후보가 아니다. 이런 경우에는 아래 중 하나를 해야 한다.

- 사용 조건을 더 명확하게 만든다.
- 효과를 발동 즉시 이득이 보이도록 바꾼다.
- AI/CPU 사용 기준을 높여 좋은 상황에서만 쓰게 한다.
- 조합 금지로 손해 페어링을 막는다.
- 그래도 안 되면 후보에서 제외한다.

## Scan Types

### Random Scan

실제 랜덤 배정에 가까운 검사다.

- 전체 허용 로드아웃에서 무작위로 대전을 구성한다.
- 최고 승률 조합과 최저 승률 조합을 본다.
- 초능력/특수돌 단위 집계가 과하게 기울어지는지 본다.
- 60% 이상 조합이 나오면 우선 관찰 또는 금지 후보로 둔다.

### Focused Scan

의심 조합만 더 많이 돌리는 검사다.

- random scan의 상위/하위 조합을 대상으로 한다.
- 표본 부족으로 튄 결과인지 확인한다.
- 특정 초능력 또는 특수돌을 살릴 수 있는 페어링을 찾을 때 사용한다.

### Value Scan

실제로 쓸 가치가 있는지를 보는 검사다.

- 같은 seed로 정상 게임, 초능력 비활성 게임, 특수돌 비활성 게임을 비교한다.
- `powerLiftAvgMargin`, `powerLiftWinDelta`, `stoneLiftAvgMargin`, `stoneLiftWinDelta`를 본다.
- 승률이 평범해도 lift가 반복적으로 음수면 후보 또는 사용 정책을 수정한다.

### Side Split Scan

카드가 선공일 때만 좋거나 후공일 때만 좋은지를 보는 검사다.

- 각 초능력/특수돌을 `black` 배정과 `white` 배정으로 나눠 본다.
- `blackScoreRate`, `whiteScoreRate`, `sideDelta`를 함께 기록한다.
- `sideDelta`가 절대값 10%p 이상이면 관찰 후보로 둔다.
- `sideDelta`가 절대값 15%p 이상이면 패치 또는 조합 금지를 우선 검토한다.
- 전체 흑/백 승률이 안정적이어도 특정 카드의 side split이 튀면 밸런싱이 끝난 것으로 보지 않는다.
- 같은 seed에서 흑백 로드아웃을 바꿔 돌리는 side-adjusted run을 기본으로 삼는다.

## Balance Targets

아래는 출시 후보로 볼 때의 대략적인 기준이다. 절대 규칙은 아니지만 벗어나면 이유를 문서에 남긴다.

- 최고 로드아웃 승률: 60% 미만 권장
- 최저 로드아웃 승률: 40% 이상 권장
- 초능력 집계 승률: 대체로 45~55% 권장
- 특수돌 집계 승률: 대체로 45~55% 권장
- value scan lift: 명백한 음수 후보가 없어야 한다.
- 선공/후공 편차: 특정 조합에서 크게 갈리면 집중 검증한다.
- 카드별 sideDelta: 절대값 10%p 미만 권장, 15%p 이상이면 패치/금지 후보.

파격 모드는 일반 랭킹전보다 허용 범위를 넓게 잡을 수 있다. 다만 "상대도 사기니까 공평하다"는 이유만으로 60% 이상 조합을 방치하지 않는다.

## Patch Order

밸런싱 패치는 아래 순서로 고려한다.

1. CPU/AI 사용 기준 수정
2. 약한 조합 금지 또는 강한 조합 금지
3. 발동 조건 조정
4. 대상 수, 지속 턴, 보호 횟수 같은 수치 조정
5. 효과 구조 변경
6. 후보 제외

재미있는 효과는 가능한 한 살린다. 다만 설명이 복잡하거나 발동 결과가 직관적이지 않으면 효과 구조 변경 또는 제외를 우선한다.

## Strong Combo Policy

사기 조합은 삭제보다 배정 금지가 우선이다.

예시:

- `A 초능력 + B 특수돌`이 60% 이상이면 해당 로드아웃만 금지한다.
- `A 초능력`이 대부분 조합에서 강하면 A의 수치를 낮춘다.
- `B 특수돌`이 대부분 조합에서 강하면 B의 발동 조건 또는 사용 기준을 조정한다.
- 특정 조합이 강하지만 매우 재미있다면 카오스 전용 관찰 후보로 둘 수 있다.

금지 조합을 추가할 때는 "강해서 금지"인지 "약해서 금지"인지 문서나 커밋 메시지에 남긴다.

## Weak Combo Policy

약한 조합도 랜덤 배정에서 막는다.

유저 입장에서는 약한 조합을 받는 것도 불공정하다. 특히 아래 케이스는 금지 또는 패치 대상이다.

- 사용 시 승률이 미사용 시 승률보다 낮다.
- value scan lift가 반복적으로 음수다.
- 특정 조합의 승률이 40% 근처까지 떨어진다.
- 효과가 발동되어도 보드에서 이득이 보이지 않는다.
- AI가 좋은 사용 타이밍을 찾기 어렵다.

## Fun And Clarity Checklist

새 후보는 숫자 검증 전에 아래 질문을 통과해야 한다.

- 이름만 봐도 대략 어떤 일이 일어날지 떠오르는가?
- 발동 순간 보드에서 영향 범위가 바로 보이는가?
- 처음 보는 유저가 카드 공개 팝업만 보고 다음 행동을 이해할 수 있는가?
- 흑/백 소유권이 시각적으로 명확하게 남는가?
- 상대도 대응할 수 있는 여지가 있는가?
- 텍스트로만 설명되는 종료 점수 보너스에 과하게 의존하지 않는가?
- 반복 발동형 효과는 첫 발동 때만 카드 공개되는가?

## Visual Feedback Requirements

밸런스 검증은 수치만으로 끝내지 않는다.

- 광역 반전은 영향받은 칸에 테두리, 잔광, 파동 같은 흔적을 남긴다.
- 구멍 생성, 봉인, 보호막, 폭발, 흡입은 서로 다른 이펙트를 쓴다.
- 특수돌은 화려해도 흑/백 소유권이 먼저 읽혀야 한다.
- 로비 설명 목록에서도 특수돌의 대표 비주얼을 보여준다.
- 랜덤 효과는 어떤 결과가 터졌는지 즉시 텍스트와 시각 효과로 알려준다.

## Documentation Requirements

밸런싱을 다시 돌릴 때는 아래를 남긴다.

- 후보 풀
- 생존 후보
- 탈락 후보와 탈락 이유
- 허용 로드아웃 수
- 금지 로드아웃 수
- 최고/최저 로드아웃
- power value scan 결과
- stone value scan 결과
- 관찰 후보
- 다음 패치에서 먼저 볼 항목

숫자만 남기지 말고 판단도 남긴다. 예를 들어 `자석돌`처럼 재미 때문에 살렸지만 value scan이 흔들리는 후보는 "생존, 관찰"로 표시한다.

## Release Policy

- 새 밸런스 풀은 바로 랭킹전에 넣지 않는다.
- 먼저 친선/카오스 모드에서 공개한다.
- 실제 유저 로그로 사용률, 사용 시 승률, 미사용 시 승률, 항복률, 재대전률을 본다.
- 관찰 후보는 출시 후 먼저 패치할 수 있도록 별도 목록에 둔다.
- 밸런싱 자동 패치는 바로 적용하지 말고 추천 리포트로 시작한다.

## RC21 Final Hard Pass (2026-06-04)

출시 후보 직전 기준은 실제 런타임 카드 풀과 로비 카드 풀이 같은지 먼저 확인한다.

- 초능력 10종
- 특수돌 10종
- 최종 허용 로드아웃 62개
- 제외 로드아웃 38개
- 명시 금지 조합 36개
- 서로 다른 초능력 2개와 서로 다른 특수돌 2개가 매 게임 배정되어야 한다.

이번 패스에서는 `scoreRate = 승리 + 무승부 0.5`로 본다. 단순 승률만 보면 무승부가 노이즈로 처리되므로 카드 간 비교에는 scoreRate를 우선한다.

### Test Runs

- 1,200게임 side-adjusted self-play
  - 같은 시드의 보드/로드아웃을 흑백으로 한 번씩 바꿔 돌렸다.
  - 흑 승률 46.6%, 백 승률 51.2%, 무승부 2.2%
  - 평균 흑 마진 -0.68
  - 진행 중단 0건
  - 초능력 중복 배정 0건
  - 특수돌 중복 배정 0건
- 최종 금지 반영 후 300게임 smoke
  - 흑 승률 49.0%, 백 승률 49.3%, 무승부 1.7%
  - 평균 54.0수
  - 진행 중단 0건
  - 초능력 중복 배정 0건
  - 특수돌 중복 배정 0건

### Card-Level Result

1,200게임 기준 카드 단위 scoreRate는 대체로 출시 후보 범위 안에 있다.

- 초능력: 44.8% ~ 56.3%
- 특수돌: 42.0% ~ 59.8%
- 모든 초능력/특수돌의 사용률은 92% 이상이었다.
- 사용 후 scoreRate가 배정 scoreRate보다 크게 낮은 카드는 없었다.

관찰 후보:

- `대폭발돌`: scoreRate 59.8%, used scoreRate 60.5%. 재미를 위해 유지하지만 출시 후 먼저 본다.
- `흡혈돌`: scoreRate 42.0%, used scoreRate 43.4%. 약한 편이지만 조합 금지 후 smoke가 안정적이라 유지한다.
- `역전세`: scoreRate 44.8%. 뒤처진 상황에서만 쓰는 카드라 raw winRate는 낮게 보일 수 있다.

### Final Patches

- `운명의 한 수`
  - 종료 보너스 +5에서 +8로 상향.
  - 표식 돌에 보호막 1회를 함께 부여.
- `소유권 폭주`
  - 다음 착수 후 추가 반전 한도를 3개에서 4개로 상향.
- 조합 금지
  - 사기 조합뿐 아니라 받아서 불리한 약한 조합도 금지한다.
  - 마지막 1,200게임에서 68% 이상 또는 32% 이하로 튄 조합은 추가 금지했다.

### Final Banned Outlier Examples

강해서 금지:

- `연쇄번개 + 대폭발돌`
- `3x3 일식 + 흡혈돌`
- `가장자리 해일 + 거울돌`
- `블랙홀 + 거울돌`
- `착수 봉인 + 암살돌`
- `착수 봉인 + 잭팟돌`

약해서 금지:

- `역전세 + 흡혈돌`
- `역전세 + 자석돌`
- `블랙홀 + 도미노돌`
- `운명의 한 수 + 잭팟돌`
- `연쇄번개 + 분열돌`
- `착수 봉인 + 분열돌`

### Release Judgment

RC21은 카오스 모드 기본 공개 후보로 둘 수 있다.

아직 랭킹전 고정 밸런스로 보기는 이르지만, 현재 기준에서는 말도 안 되는 자동 승리 조합이나 배정 즉시 손해인 조합은 금지 풀로 걸러진 상태다. 다음 밸런싱은 실제 유저 로그 기준으로 먼저 `대폭발돌`, `흡혈돌`, `역전세`를 본다.

## RC22 Side-Split Hard Pass (2026-06-04)

이번 패스부터 카드별 흑/백 분리 승률과 scoreRate를 정식 지표로 추가한다. 전체 승률이 안정적이어도 특정 카드가 선공 또는 후공에만 과하게 좋으면 밸런싱 완료로 보지 않는다.

### Test Runs

총 33,000게임을 side-adjusted self-play로 돌렸다.

- 같은 seed의 보드/로드아웃을 흑백으로 바꿔 한 번씩 돌렸다.

### Release Judgment

RC22는 카오스 모드 출시 후보로 둘 수 있다.

이번 기준에서는 자동 승리급 조합, 배정 즉시 손해 조합, 선공/후공 편향 조합이 모두 금지 또는 조정됐다. 랭킹전 고정 밸런스는 실제 유저 로그를 본 뒤 결정하고, 다음 패치에서는 `흡혈돌`, `블랙홀`, `분열돌`을 먼저 관찰한다.
